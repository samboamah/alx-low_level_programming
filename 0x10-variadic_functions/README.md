# Variadic functions
Project done during **Full Stack Software Engineering studies** at **[ALX AFRICA](https://www.alxafrica.com)**. It aims to learn about variadic functions in **C language**.

## Technologies
* C files are compiled using `gcc 11.2.0.`
* C files are written according to the C90 standard
* Tested on Ubuntu 22.04 LTS.

## Files
All of the following files are programs written in C:

| Filename | Description |
| -------- | ----------- |
| `0-sum_them_all.c` | Returns the sum of all its parameters |
| `1-print_numbers.c` | Prints numbers |
| `2-print_strings.c` | Prints strings |
| `3-print_all.c` | Prints anything |
| `100-hello_holberton.asm` | Assembly that prints `Hello, Holberton` |
